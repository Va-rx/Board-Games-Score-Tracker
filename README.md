# Board Games Score Tracker
Web application designed for tracking and managing board game results, featuring a dynamic schema for creating and sharing custom scoring columns.

## Technology
1. Java
2. Spring Boot
3. Hibernate
4. Lombok
5. Docker
6. PostgreSQL

## Key Features
1. Core Domain Model
2. Well working API for Users, Groups, Games with appropriate status codes and messages

## Roadmap
1. Rest of the endpoints: e.g: Creating Presets
2. JWT-based auth
3. Tests
4. Frontend
